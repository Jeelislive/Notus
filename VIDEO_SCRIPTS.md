## Recording Checklist

- [ ] Use a clean browser profile (no extensions visible in toolbar)
- [ ] Set browser zoom to 90% for more screen real estate
- [ ] Use a demo account with realistic meeting data pre-loaded
- [ ] Record at 1920×1080 (or 2560×1440 for Retina)
- [ ] Microphone: quiet room, USB mic preferred
- [ ] Cursor highlighting: enable in system settings or use Cursor Highlight app
- [ ] Edit out all loading spinners > 2s (speed up or cut)
- [ ] Add subtle background music at -20dB
- [ ] Add lower-third text overlays for each feature name
